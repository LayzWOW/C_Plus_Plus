## Beginner Projects

1) [x] Simple Calculator:
      - performs simple calculation (addition, subtraction, multiplication, division)
      - user input validation and function implementation

2) [x] Number Guessing Game:
      - user guess a random number generated by the computer
      - control flow statements

3) [] To-Do List Manager:
      - CLI application where user can create, read, update, delete task
      - data structures and I/O to save files

4) [] Tic-Tac_Toe Game:
      - a two-player game on a 3X3 grid
      - array manipulation, game logic, determine winning conditions

5) [] Student Database System:
      - to store and manage student records (names, grades, class, etc)
      - data structures and basic file handling

## Intermediate Projects

1) [] Phone Book Application:
      - store and organize contact information, allowing users to search and sort contacts
      - databases for storage and (advance) data structures such as Binary Search Trees (BST)

2) [] Text Editor:
      - allows user to create, open, edit and save text files
      - using GUI libraries like Qt or GTK+ to add graphical user interface

3) [] Snake Game:
      - build using SDL graphical interface 
      - great for practicing game loop, collision detection, and OOP principles.

4) [] Library Management System:
      - for managing library resources and user accounts, including check in/out and fine tracking
      - heavy usage on OOP, databases (like MySQL), and file handling

5) [] Image Processing Tool:
      - using the OpenCV library to perform simple image manipulations like blurring, color
        conversion, or shape detection

## Advanced Projects

1) [] Custom Game Engine:
      - build a 2D or 3D renderer using low-level API like OpenGL or Vulkan
      - involves complex mathematics(linear algebra), data structures, and performance
        optimization

2) [] Networked Chat Application:
      - create a client-server application for real-time messaging
      - introduces networking concepts, socket programming(TCP/IP), and multithreading

3) [] Web Browser:
      - build a basic web browser using framework such as Qt creator to handle the GUI
        networking APIs, and HTML/CSS parsing

4) [] Online Voting System:
      - secure system that uses cryptography and database management to conduct online elections
      - heavily relies on security measures and networking

5) [] Implement an STL container:
      - mastering core C++ features and performance, implement own version like std::vector or std::set 
        to understand their inner workings
